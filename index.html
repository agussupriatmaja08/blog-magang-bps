Project

Proyek Saya

Berikut merupakan proyek hasil dari pembelajaran saya, baik secara mandiri maupun dalam kerja kelompok

Book restApi-Js

Book restApi-Js adalah program REST API yang menggunakan framework HAPI. Pengguna dapat melakukan operasi CRUD pada data buku.

Basic Calculator App

Basic Calculator App adalah program kalkulator sederhana yang menggunakan bahasa pemrograman JavaScript.

Simple CRUD for Dealer

Simple CRUD for Dealer adalah sistem sederhana berbasis PHP yang dirancang khusus untuk dealer sepeda motor. Memungkinkan pengguna untuk melakukan pembelian produk, sementara administrator dapat melakukan CRUD pada produk dengan tambahan fitur login dan logout untuk membedakan antara pengguna biasa dan administrator.

SuitGame

Suit Game dibuat menggunakan bahasa pemrograman JavaScript. Dalam permainan ini, pengguna akan berhadapan dengan komputer. Dalam permainan ini, pengguna memilih salah satu dari tiga simbol, yaitu batu, gunting, atau kertas, untuk bersaing dengan pilihan acak komputer.

Sistem Management Rumah Sakit

Sistem Manajemen Rumah Sakit adalah aplikasi CRUD yang dibangun dengan menggunakan framework Laravel. Sistem ini memungkinkan administrator untuk melakukan login, logout, dan melakukan berbagai operasi CRUD, termasuk CRUD Data Pasien, CRUD Surat Rujukan Sakit, CRUD Data Ruangan Rumah Sakit, dan CRUD Data Rawat Inap.

MahasiswaDB

MahasiswaDB adalah sistem CRUD (Create, Read, Update, Delete) sederhana untuk data mahasiswa yang diimplementasikan menggunakan runtime Node.js.

Game Journey Sun Go Kong to the west

Game Journey Sun Go Kong to the West adalah proyek permainan yang dibuat menggunakan bahasa pemrograman Java, dikembangkan dalam lingkungan Greenfoot. Tujuan dari permainan ini adalah untuk membantu Sun Go Kong dalam perjalanannya ke Barat untuk mengalahkan raja terakhir.

Developer : Gede Agus Supriatmaja, Made Alit Juniska, Zulvan Avito Anwari, dan Ni Ketut Rika Suryani

Database Sekolah

Database Sekolah adalah sistem CRUD yang dirancang untuk mengelola data siswa dan guru sekolah. Sistem ini diimplementasikan menggunakan bahasa pemrograman PHP. Selain dari fungsionalitas CRUD, sistem ini juga menyertakan mekanisme login dan logout untuk meningkatkan keamanan.
